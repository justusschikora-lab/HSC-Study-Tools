How to use this

Edit profile — Tap Edit profile to set your name, username, bio, the note bubble above your photo, follower & following counts, profile picture and the DM badge number.

Add posts — Tap the + at the top-left (or Create on the empty state). You can pick several photos at once; the grid grows on its own and the post count updates.

Add highlights — Tap the New circle to choose a cover photo, crop it, and give it an emoji or title.

Remove things — Tap a post to open it and hit Remove; tap a highlight to edit or remove it.

Saving — Everything saves automatically on this device. The lock icon by your name is a manual quick-save.

Start over — In Edit profile, use Reset profile or Start from scratch.
